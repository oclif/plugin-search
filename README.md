# Usage

<!-- usage -->

```sh-session
$ npm install -g @oclif/plugin-search
$ @oclif/plugin-search COMMAND
running command...
$ @oclif/plugin-search (--version)
@oclif/plugin-search/1.2.35 linux-x64 node-v20.19.5
$ @oclif/plugin-search --help [COMMAND]
USAGE
  $ @oclif/plugin-search COMMAND
...
```

## `@oclif/plugin-search search`

Search for a command.

```
USAGE
  $ @oclif/plugin-search search

DESCRIPTION
  Search for a command.

  Once you select a command, hit enter and it will show the help for that command.
```

_See code: [src/commands/search.ts](https://github.com/oclif/plugin-search/blob/1.2.35/src/commands/search.ts)_
